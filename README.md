# Landing Page — Barbearia

Landing page estilosa e responsiva criada como projeto demonstrativo para o segmento de barbearias.

**[Ver site ao vivo](https://salomaostudart.github.io/landing-barbearia/)**

## Funcionalidades

- Design escuro com tons bronze, estética masculina
- Lista de serviços com preços e duração
- Galeria de fotos dos trabalhos
- Formulário de agendamento com seleção de barbeiro, serviço, data e horário
- Horário de funcionamento
- Seção de localização
- Menu hamburger para dispositivos móveis
- Animações suaves ao rolar a página
- Acessibilidade (ARIA labels, skip-to-content)
- Meta tags Open Graph para compartilhamento em redes sociais

## Tecnologias

- HTML5 semântico
- CSS3 (variáveis CSS, grid, flexbox, media queries)
- JavaScript vanilla (sem dependências externas)
- Imagens otimizadas via Unsplash CDN

## Performance

- Zero dependências externas (sem frameworks, sem bibliotecas)
- CSS e JS inline (single file, sem requests adicionais)
- Imagens otimizadas com parâmetros de compressão
- Carregamento rápido em qualquer dispositivo
